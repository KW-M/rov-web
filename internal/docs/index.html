To edit the extension options: 2. Go to the extensions manager in the Blue OS Web UI 2. Enable pirate mode 3. Find the ROV-Web extension under the "Installed Extensions" tab 4. Click the edit link for the ROV-Web extension 5. Under the "Custom Settings" tab, you can edit the extension options. 6. Settings is a json object with the JSON field "Env" containing an array of enviroment variables as quoted strings. Enviroment variables use the format "KEY=VALUE" where KEY is the name of the variable and VALUE is the value of the variable. Don't add extra spaces or quotes! 7. Click the "Save" button to save your changes Example custom settings: 

            {
                "Env": [
                    "ROV_NAME=My ROV",
                    "ROV_CONTROL_PASSWORD=1234ABC",
                ]
            }
